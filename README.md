1. Project Title
# Vibe Streams
Music Streaming Android Application

2. Description

A short paragraph describing the project.

Vibe Streams is an Android-based music streaming app that allows users to browse, search, and play songs. Users can create, edit, and share playlists, while an admin panel manages song uploads and edits using Firebase Authentication and Firestore for backend.

3. Features

List the main features clearly:

## Features

### User Module:
- User registration and login
- Home page with Trending, New Releases, and Recommended songs
- Search songs by title or artist
- Create, edit, delete, and share playlists
- Full-screen music player and mini-player for continuous playback

### Admin Module:
- Admin login with fixed credentials
- View all uploaded songs
- Add new songs (title, artist, image URL, audio URL)
- Edit or delete existing songs

4. Technologies Used
## Technologies Used
- Android Studio
- Java
- Firebase Authentication
- Firebase Firestore
- XML for UI
- RecyclerView for lists

5. Installation / Setup

Explain briefly how to set up the project if someone wants to run it.

## Installation

1. Clone the repository.
2. Open the project in Android Studio.
3. Connect your Firebase project:
   - Add `google-services.json` in `app/` folder.
   - Set up Firestore and Authentication in Firebase console.
4. Build and run the app on an Android device or emulator.

6. Usage

Explain briefly how the app works.

## Usage
- Register a new account to log in.
- Browse songs from the Home page categories.
- Search for songs by title or artist.
- Create and manage playlists in the Library.
- Admin can log in to manage songs.
